### Troubleshooting Microphone
```
Problem: Microphone not detected

  1. Refresh the page
  2. Check browser permissions
  3. Test microphone in browser settings
  4. Try Chrome/Edge if using Firefox

Problem: Noise in background

  1. Use quiet environment
  2. Reduce fan/AC noise
  3. Close background applications
  4. Use microphone headset for better clarity

Problem: Recognition not working
Solution:
  1. Speak louder and clearer
  2. Check microphone volume
  3. Restart browser
  4. Try different browser
```

## üÜò Getting Help

### Common Issues

**Issue: Voice not being recognized**
- Solution: Speak louder, clearer, in quiet room
- Check microphone in browser settings
- Try refreshing the page

**Issue: Images not loading**
- Solution: Check internet connection
- Pexels API might be rate-limited (wait 5 min)
- Fallback images will still work

**Issue: PowerPoint not downloading**
- Solution: Check browser download settings
- Try different browser
- Check file permissions

**Issue: File too large**
- Solution: Normal for embedded images
- Compress images in PowerPoint if needed
- Use "Reduce File Size" feature

### Contact Support
- Check browser console for errors (F12)
- Verify all components running
- Test with simple 1-2 word slides first
